# מערכת הצפנה - פרויקט קריפטוגרפיה

## התקנה והפעלה

### דרישות מקדימות
- Python 3.x
- PyCharm (מומלץ)

### שלבי התקנה

1. פתח את תיקיית הפרויקט ב-PyCharm
2. התקן את כל הספריות הנדרשות:
```
pip install -r requirements.txt
```
3. הרץ את הפרויקט:
```
python app.py
```

## על הפרויקט

מערכת הצפנה הכוללת שני צפנים:
- **Reversed Substitution** — החלפת אותיות לפי אלפבית מבוסס מפתח
- **Climbing Key** — הזזה עולה (כל אות מוזזת יותר מהקודמת)

כולל: הצפנה, פענוח, פריצה אוטומטית, ומערכת הרשאות עם bcrypt.
